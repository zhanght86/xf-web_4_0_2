使用常见问题

1. 小富机器人功能？

面向大中型企业的私有部署智能问答交互产品。是一款更懂你的机器人!

2. 小富机器人的重要性？

小富的设计融合各个行业领域的需求推出分场景设计，对相同场景的业务提供多样化的支撑，提供多样化的知识加工。

3. 小富机器人的目标用户？

大中型企业，政府的知识条目都很复杂，小富机器人有很强大的业务类目与知识加工功能，能帮助客户快速的加工知识，-因此小富是大中型企业客户的理想工具

4. 小富机器人设计理念？

神州泰岳的小富机器人是一款人工智能界的创新产品，它提供了业界首款面向场景的智能问答机器人。 小富机器人从设计之初一直秉承着极致创新的产品理念，为用户打造爱不释手的产品。 小富的设计融合各个行业领域的需求,在产品设计上采用了业界领先的语义认知技术，而且在交互体验上对企业用户的实际操作过程做了细致考虑。 目前，小富机器人已经为政府，银行等数家大型企业提供智能问答服务。

5. 小富机器人的产品定位和价值？

(1) 小富主要定位于为企业、政府、个人用户提供智能问答解决方案。

(2) 小富的价值主要体现在：业内首款面向场景设计的交互机器人，采用目前业内最先进的算法，提供了整体性业务建模，场景化的知识表达体系，以及具有碎片化知识加工的功能。
